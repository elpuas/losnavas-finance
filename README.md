
# Los Navas Finance

A personal family finance app to track income, expenses, and calculate remaining balance.

Private project developed by ElPuas Digital Crafts.

## Tech Stack

- React (Vite)
- Supabase

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```
