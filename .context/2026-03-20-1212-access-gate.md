# Change Log

## Task summary

Added a lightweight client-side access gate controlled by an environment variable.

## Files created or modified

- `/src/app/components/AccessGate.tsx`
- `/src/app/App.tsx`
- `/vite.config.ts`
- `.context/2026-03-20-1212-access-gate.md`

## Description of changes

- Added `AccessGate` to show a minimal full-screen entry screen before rendering the app
- Read the access phrase from `import.meta.env.LOS_NAVAS_APP_ACCESS`
- Compared the submitted phrase against the configured environment variable
- Stored remembered access in `localStorage` under `navas_access`
- Wrapped the app root with the access gate without changing routes or business logic
- Updated Vite config to expose `LOS_NAVAS_*` environment variables to the client

## Assumptions

- This gate is only intended as a lightweight client-side barrier for a private internal tool
- If `LOS_NAVAS_APP_ACCESS` is missing, the gate should not block access

## Verification

- `npm run build` passed successfully
