# Change Log

## Files created or modified

- `/index.html`
- `/public/manifest.json`
- `/public/icons/icon-192.png`
- `/public/icons/icon-512.png`

- Added the manifest link and theme color meta tag to the main HTML `<head>`

- Added install icons in `public/icons` at `192x192` and `512x512`
